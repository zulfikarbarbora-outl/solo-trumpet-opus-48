# solo-trumpet-opus-48

Solo trumpet multisamples — sustained synth trumpet tones at 8 chromatic pitches (Akai AX80 'Synth Trumpet' pack 17233 by modularsamples, CC0) + 3 real trumpet one-shots at C (James_KuKu 'Brass C Note' pack 41487, CC0).

## About

This repo holds **48 kbps Opus previews** for fast streaming in the web DAW.
Opus at 48k delivers near-CD quality at ~35% the size of MP3 128k.

- **Library**: Solo Trumpet
- **Files**: 11 Opus previews
- **Bitrate**: 48 kbps VBR
- **Container**: Ogg Opus (.opus)
- **License**: CC0-1.0
- **Source**: https://freesound.org/people/modularsamples/packs/17233/
- **Branch**: `main`
